Heroku Spring Boot PostgreSQL :

This is a Demo of a MVC application, using Spring boot framework. This application is connected to a PostgreSQL database by a JDBC driver. These two components are hosted on Heroku.

Here is the list of used tools :

  • Java 1.8
  • Maven 3.5.2
  • PostgreSQL 10
  • Heroku CLI

The aim of this application is to list godfather in the Mafia
